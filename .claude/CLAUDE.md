# TelcoNow Project

A web application for TelcoNow, a fictional Australian telco. Includes a public homepage,
a login page, and an authenticated customer dashboard.

## Tech Stack

| Layer | Tool |
|---|---|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript (strict mode) |
| Styling | Tailwind CSS |
| CMS | Contentful |
| Auth | NextAuth.js v5 |
| Hosting | Vercel |
